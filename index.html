Projects

ServiceNow Hardware Updater

A tool that streamlines updating ServiceNow Hardware Assets at device EOL. Given only a Computer Name, the Serial Number of the old device, and the Serial Number of the new device, the old record is retired and the new record is updated to reflect the old record (While also allowing for edits to be made).

Application Installer Interface

An interface that enables technicians to install applications hosted through Microsoft SCCM to remote machines quickly, easily, and/or in bulk. The responsive menus allow the tech to enter a list of Computer Names, search available apps, push an install or uninstall, and check the installation status.

Network Printer Automation

A tool that automates connecting end-users' network-connected printers after device upgrade. It generates a list of all connected printers for all User accounts on the old device. It formats that list into a .vbs print script which is then saved to the startup folder of the new device so that all printers will be automatically added at login.
